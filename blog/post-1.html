The initial version of fort has arrived!

Announcing fort v0.1.0

I'm very happy to announce that the initial version of fort has been released.

Where to get it

Fort is open source software so you can download it here: https://github.com/fortlang/fort


There you can find an overview of the fort language itself, source code for the compiler, test cases, and example programs written in fort.

Word of warning

There isn't currently much in the way of documentation and as you can tell from the version number this should be considered alpha (or pre-alpha!) level software.


However, all of the basic constructs are implemented so you can get a feel for writing code in fort and perhaps try it out on some examples of your own. Any contributions/feedback would be much appreciated!


Every project has to start somewhere and hopefully from these humble beginnings fort can grow into something truly useful.
